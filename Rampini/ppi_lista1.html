A Diferença entre Internet e Web

Internet → é a rede mundial de computadores. É a infraestrutura física e lógica que conecta milhões de dispositivos (cabos, roteadores, servidores, etc.).

Web (World Wide Web) → é um serviço que funciona dentro da Internet, usado para acessar páginas e sites por meio de navegadores (como Chrome ou Firefox).

História da Internet

  • Década de 1960:

    Surge nos Estados Unidos o projeto ARPANET, criado pelo Departamento de Defesa para conectar computadores de pesquisa militar e universitária. → Em 1969, quatro universidades americanas se conectam pela primeira vez

  • Anos 70

    São desenvolvidos os protocolos TCP/IP (Transmission Control Protocol / Internet Protocol), que permitiram a comunicação entre diferentes redes. → Em 1983, o TCP/IP se torna o padrão, e o termo “Internet” começa a ser usado.

  • Anos 1980:

    Expansão para universidades e centros de pesquisa no mundo inteiro. → Em 1988 é criado o primeiro provedor de e-mail.

  • Anos 1990:

    A Internet chega ao público geral com a criação de navegadores e provedores comerciais. → O número de usuários cresce rapidamente.

História da Web

  • 1989–1990

    O cientista Tim Berners-Lee, no CERN (Suíça), cria a World Wide Web, com três pilares fundamentais:

    1. HTML (linguagem para criar páginas),
    2. HTTP (protocolo de transferência de páginas),
    3. URL (endereços dos sites).

  • Anos 70

    São desenvolvidos os protocolos TCP/IP (Transmission Control Protocol / Internet Protocol), que permitiram a comunicação entre diferentes redes. → Em 1983, o TCP/IP se torna o padrão, e o termo “Internet” começa a ser usado.

  • Anos 1993:

    Surge o Mosaic, primeiro navegador com interface gráfica — origem dos navegadores modernos

  • Anos 2000:

    Expansão de sites, buscadores, redes sociais e e-commerce. → Aparece o conceito de Web 2.0, com foco em interação e conteúdo gerado por usuários
