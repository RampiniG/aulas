Conteudo principal da pagina

HTML não é uma linguagem de programação, HTML é lingugem de estrutura ela serve para construir uma página simples = o esqueleto da página


As tags essnciais:
!DOCTYPE html -> Tipo do Documento
html -> "Inicio da Página"
head -> Cabeça da Página
meta charset="UTF-8" -> Basicamente o Dicionário
title> -> Titulo da página, oque está entro os titles é oque será escrito na aba no navegador
body -> Corpo da página



Rodapé da página -> Sempre ficará na parte inferior da página
